IndividuMonstre

class IndividuMonstre(var id: Int, var nom: String, val espece: EspeceMonstre, var entraineur: Entraineur? = null, expInit: Double)

Représente une instance de monstre individuel dans un système de jeu. Chaque monstre est basé sur une espèce donnée, mais possède ses propres caractéristiques, statistiques et attributs qui évoluent avec l'expérience.
