Technique

class Technique(val id: Int, val nom: String, val precision: Double, var multiplicateurDePuissance: Double, val estBuff: Boolean, val estDebuff: Boolean, val faireDegats: Boolean, val estSpecial: Boolean, val elementTechnique: Element)

Décrit une technique utilisable par un monstre en combat.

Une technique peut infliger des dégâts, être spéciale (basée sur l’attaque spéciale) ou appliquer des effets (buff/debuff). Elle possède une précision et un multiplicateur de puissance, ainsi qu’un élément utilisé pour calculer les affinités.
