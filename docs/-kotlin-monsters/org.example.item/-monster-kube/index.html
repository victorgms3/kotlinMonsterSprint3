MonsterKube

class MonsterKube(val id: Int, val nom: String, val description: String, var chanceCapture: Double) : Item, Utilisable

Objet consommable permettant de tenter la capture d’un monstre sauvage.

La probabilité de capture dépend des PV restants de la cible et du paramètre chanceCapture. Implémente Utilisable pour être utilisé en combat.
