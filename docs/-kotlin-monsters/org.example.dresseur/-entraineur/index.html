Représente un entraîneur dans le contexte du jeu.

Un entraîneur est responsable de gérer une équipe de monstres, une boîte pour stocker des monstres supplémentaires et un sac contenant des objets appelés MonsterKubes. L'entraîneur a également une somme d'argent associée.
